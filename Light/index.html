Click Button to Change Image with JS

JavaScript can change the value of HTML image attribute with onclick.

Here, clicking a button produces different images of a light bulb. All you need to do is have the image files uploaded into the project folder along with this index.html file and it will work in a browser.

The onclick JavaScript event occurs when the user clicks on an element. It runs a specified line of code when you click a HTML object that has the onclick attribute. The JavaScript onclick functions can be triggered by object.onclick or object.addEventListener.

Here is the JS code running behind this:

1. use "button onclick="document.getElementById('myImage').src='pic_bulbon.gif'" tags around the "Turn on the light" button

2. insert code: img id="myImage" src="pic_bulboff.gif" style="width:100px" -- this is so the bulb's default state when not clicked is "off"
If you changed this so the source was "pic_bulbon.gif" the default state would be the bulb is turned on.
3. use "button onclick="document.getElementById('myImage').src='pic_bulboff.gif'" tags around the "Turn off the light button
